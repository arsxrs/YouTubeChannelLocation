### The extension for web browsers shows info about the YouTube channel's location(country) in the video description.
![image](https://user-images.githubusercontent.com/2439426/173799328-8d47ff22-6a36-4657-8153-dabe173faff8.png)

Available at stores:
- [Chrome](https://chrome.google.com/webstore/detail/youtube-channel-location/lagfgccfpnemkkekoonccjidinipogpd)
- [Mozilla](https://addons.mozilla.org/ru/firefox/addon/youtube-channel-location/)
- [Opera](https://addons.opera.com/en/extensions/details/youtube-channel-location/)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/youtube-channel-location/mnlpjaiabgonppabdbjjlielocdboigg)
